## Vue CLI
    CLI: command-line-interface,
    通过CLI可选择项目的配置创建项目
    Vue CLI内置了webpack相关配置

#### 安装及使用
全局安装
```bash
$ npm i @vue/cli -g
$ vue --version       // 查看安装的版本
```
旧版本升级
```bash
$ npm update @vue/cli -g
```
创建vue项目
```bash
$ vue create projectname
```
